📚 Version 1.3.0

📚 Version 1.2.0

Version 1.2.0 (30.01.2024)

Hinzugefügt

  • SQLite database instead of CSV files
  • New table `username_history` for tracking username changes
  • `get_username_history` function to query username history
  • Automatic logging of username changes with timestamp

Geändert

  • Changed `DATABASE_FILE` from `.csv` to `.db
  • Optimized `save_to_database` function for SQLite
  • Adjusted database structure for better history tracking
